Everything is a loop, never a single shot. Every pipe output where there's an artifact needs a process for replication and reproduction — like presets on a synthesizer.

**Why:** The system had SOPs as inert documents and modules as fire-and-forget. No connection between "artifact produced" and "process governs it." The modular synthesis metaphor: the synth ships with presets, not just a manual.

**How to apply:** When designing any new module or artifact output, always ask: "what patch governs this output? what signal does it emit? what SOP receives it?" If the answer is "nothing," that's a blind spot — log it.
