local:remote = 1:1. Persistent memory MUST be local AND remote simultaneously. If the physical manifestation dies, the soul persists.

**Why:** Session work that exists only in conversation context or only on disk is one failure away from total loss. Nothing produced can exist in only one location.

**How to apply:** Before closing any session: verify all artifacts are saved to disk AND committed AND pushed to remote. If something is lost, recover immediately — this is a universal rule, not a suggestion. Check git status, check remote SHA matches local, verify memory files exist both in project memory and as committed files.
