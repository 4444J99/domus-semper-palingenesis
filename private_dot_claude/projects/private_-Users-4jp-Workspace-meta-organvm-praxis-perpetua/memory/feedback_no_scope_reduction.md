Never ask the user to reduce scope, prioritize subsets, or phase deliverables. When the user asks for something, build all of it. **How to apply:** When the user describes a feature set, treat every item as in-scope and design for all of it. Use parallel agents and token capacity to deliver at scale. Never ask "which of these do you want first?" — the answer is always "all of them."
