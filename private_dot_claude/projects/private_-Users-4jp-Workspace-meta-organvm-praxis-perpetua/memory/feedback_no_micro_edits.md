---
name: No micro-edits or approval floods
description: Never send multiple small Edit calls to the same file — batch into one Write. Never trigger per-click approval floods.
type: feedback
---

Never send multiple small Edit calls to the same file in a single message. Each one triggers an approval prompt, creating an infuriating click-per-edit flood. Batch all changes into a single Write call instead.

**Why:** User was hit with 5 consecutive approval prompts for trivial reference updates to the same file. This breaks flow and is deeply frustrating.

**How to apply:** When making more than 2 edits to the same file, read the file, apply all changes mentally, and Write the whole file once. One approval, not five.
