`~/resistance--certain-none/` is a transient triage / scratch directory. The user
opens Cursor and Claude Code sessions here to QA disparate work threads; the
directory itself is not a project home for any artifact.

**Why:** observed naming convention (`resistance--certain-none` is a generated
ontological-renamer slug, not a domain) plus the file pattern: only ephemeral
inputs (Untitled stub, transcript dumps) accumulate here.

**How to apply:** at session close-out for any work in this dir, every durable
artifact must be relocated to its proper organ home (`~/Workspace/<organ>/…`
or `~/.claude/plans/`). Date-prefixed transcript dumps go to `~/Workspace/`
directly (existing convention — see siblings like
`2026-04-27-223423-previous-relay-hand-wring.txt`). Anything left in
`~/resistance--certain-none/` after a tending pass should be the manifest
itself, awaiting one final move.
