Design doctrine for this project's materials: **Shibui/Shibumi** — profound understated beauty where the surface is simple but rewards sustained attention with layers of depth.

**Why:** The user explicitly sourced this doctrine from research on "depth intrigued by simplicity." Related concepts: Yugen (unseen depth), Ma (emptiness as potential), Simplexity (simple interface masking complexity). The pitch target works in 3D animation, so visual sophistication is non-negotiable.

**How to apply:** Ultra-light typography (weight 200/300), generous negative space, glassmorphism (backdrop-filter blur through ambient background layers), CSS 3D orbital geometry, staggered entrance animations, minimal color (monochrome + one gradient accent). Fewer words per slide. The background IS the art — floating orbs and wireframe orbital rings create living depth. Never flashy — always breathing, organic, subtle.
