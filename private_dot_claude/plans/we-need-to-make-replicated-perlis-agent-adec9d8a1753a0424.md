## The seven patient plays

### Play 1 — MetaMask MASK airdrop fleet farming (Q3-Q4 2026 TGE)

**NAME:** Multi-wallet MASK eligibility farm via Linea Surge / MetaMask Card / Swaps and Bridge — 5 to 9 month horizon. Consensys IPO is the structural forcing function; MASK is widely modeled at 50B–80B FDV with a 5–8% community allocation.

**LOOP:**
- Agent provisions N=200–800 wallets across distinct browser fingerprints + residential proxies (free tier).
- Each wallet hits MetaMask Rewards point thresholds: minimum swap volume per wallet ($1,250 buys 250 points; cap is 50,000 points per wallet at $250k cumulative volume); MetaMask Card spend (US users); bridge volume; and MetaMask Swaps + Bridge are confirmed loyalty-bonus eligible. Sustained-activity multiplier rewards transactions in 6 different months.
- Capital constraint solved by routing the same USDC/ETH float through wash swaps — only the fee delta is consumed (~0.1–0.3% per trip on Linea), not the principal. With a starting $50–$100 borrowed/bridged, generating $1,250 per wallet of "volume" costs $4–$15 in net fees per wallet. 200 wallets at $5 net = ~$1,000 burn for ~$10,000+ expected airdrop value at central estimates.
- Sybil-defeat layer: rotate gas-fund sources, vary timing windows, vary in/out paths, never reuse the same EOA+IP+useragent triple.

**30-day:** $0–200 (actual cash drain; the fleet is BURNING money to register volume).
**90-day:** $0 — TGE has not happened. Pure positional accrual.
**180-day:** $30,000–$200,000 expected value. Per-wallet historical airdrop comparable: Arbitrum $1k–$5k, Optimism $500–$3k, Linea sybil-filtered ~$200–$2k. MASK is being architected as the largest wallet airdrop ever (Consensys IPO context). Conservative central: 150 of 800 wallets clear sybil at $300 average = $45k. Aggressive central: 400 wallets at $800 = $320k.

**WHY MISSED:** 30-day filter eliminates this entirely. There is no possible return on day 30 — TGE is not even hinted before Q3. The 30-day scout sees only "burn $1k for $0," which fails their criterion.

**FALSIFIER:** Consensys announces (a) snapshot already taken pre-2026-05-04, (b) sybil filtering via Proof-of-Humanity that requires biometric verification (Linea v2 used Human Passport — assume MASK does too; this is the central risk), or (c) airdrop is keyed to MetaMask SRP age >12 months, in which case the 200 fresh wallets are dead. Mitigation: layer 50–100 wallets onto SRPs that pre-date 2025; the rest are pure farming.

### Play 3 — Programmatic SEO empire on a high-CPC AI-tooling niche (6-month rank curve)

**NAME:** Build 3,000–10,000 page programmatic SEO site on a vertical like "MCP server for [job-to-be-done]" or "[specific framework] AI SDK examples" — domains where commercial-intent CPC is $4–$15 and incumbents are slow. Ranks meaningfully at month 4–8; pays $5k–$30k/mo from month 6 onward.

**LOOP:**
- Agent generates page templates (one master template, thousands of permutations from a structured dataset — e.g., 1,200 MCP servers × 50 use-case angles = 60,000 pages, then prune to the 5,000 with non-zero search volume).
- Quality-gate each page: minimum unique value-add (pricing comparison block, code snippet, eligibility table, decision tree) so it passes 2026 Google "thin content" filters. The Omnius case shows 67 → 2,100 monthly signups in 10 months from programmatic SEO; the 4–8 month rank curve is well-documented.
- Hosting on Cloudflare Pages (free) or Vercel (free hobby). Indexed via sitemap + RSS + ping. Backlink layer via Reddit / HN / GitHub README backlinks (organic-looking, agent-distributed).
- Monetization layer 1: affiliate links to MCP hosting (Smithery, MCP-Hive), AI SDK platforms, restaking protocols. Layer 2: programmatic display via Mediavine/Raptive once at 50k sessions/mo.
- Adjacent compounding play: the canonical dataset of MCP servers / AI SDK templates becomes the JSON feed that *other people's* sites consume. Self-reinforcing canonicality.

**30-day:** $0–$50. Pages indexed; rankings forming. Long-tail traffic begins.
**90-day:** $300–$3,000/mo. Clusters of pages converging on page-one. Affiliate clicks beginning to convert.
**180-day:** $5,000–$30,000/mo recurring (i.e., $30k–$180k total over the 6-month build → 12-month tail).

**WHY MISSED:** 30-day SEO is a known impossibility. Operators chasing $10k/30 days never start the 4–8 month curve, so the curve never compounds. This is the single highest-leverage missed play.

**FALSIFIER:** Google rolls out a programmatic-content algorithmic penalty mid-2026 (HCU-style) targeting AI-template sites. Mitigation: pages must encode genuine new-knowledge — comparison tables, structured eligibility, dynamic data feeds. Pure rephrased AI slop dies; structured-data utility pages survive.
