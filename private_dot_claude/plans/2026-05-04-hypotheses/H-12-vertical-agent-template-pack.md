# H-12 — Vertical agent template pack (claude-agent-sdk for non-tech verticals)

**Category:** Translation gap

## Claim
Claude Agent SDK / LangGraph / Pydantic AI patterns are widely used by tech teams but barely deployed in non-tech SMBs (real estate brokers, accountants, insurance agents, contractors). A pack of 5–10 vertical-specific agent templates (e.g., "lead-qualifier agent for real estate," "expense-categorizer agent for accounting") at $99–249 converts to vertical SMB buyers who'd otherwise hire dev consultants.

## Why missed
- Tech-side builders don't know vertical workflows.
- Vertical SMBs don't know agent SDKs exist.
- Translation gap is wide; nobody is sitting in both seats.

## Falsifier
Dies if: vertical SMBs won't deploy AI agents themselves (need consultants); incumbents in each vertical (e.g., Follow Up Boss for real estate) already ship AI features; willingness-to-pay for templates is below floor (vertical SMBs tend to want done-for-you, not templates).

## Confirmer
Vertical SMB Reddit/Facebook groups discussing AI tools; willingness-to-pay proven by adjacent vertical software pricing ($99-499 one-time); vertical conferences / podcasts featuring AI demand.

## Payout
$99–249 × 50–150 = $5–37k. Plausible to clear $10k.

## Cost-to-falsify
Medium. ~6 hours: pick 2 candidate verticals, audit existing AI offerings in each, demand probe via vertical communities.

## Bottom-line
**Plausible but distribution-hard.** Vertical SMBs don't aggregate online cheaply. Advance to Phase 2 mid priority; favor only if a partner channel (vertical newsletter, podcast, association) emerges.
