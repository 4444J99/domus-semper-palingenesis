# H-10 — Cloudflare Workers / Durable Objects / D1 deep production guide

**Category:** Documentation gap

## Claim
Cloudflare's edge stack (Workers, Durable Objects, D1, Hyperdrive, R2, Queues) has exploded in capability but official docs are reference-style and thin on production patterns. A paid book / course / template pack at $49–99 covering "real production patterns for the Cloudflare edge stack" converts.

## Why missed
- Cloudflare moves fast; docs lag intentionally.
- Existing books on serverless target AWS Lambda; Workers-specific is sparse.
- The ideal author is a practitioner with production scars; few have published.

## Falsifier
Dies if: a strong existing book/course already dominates (probable — must check Educative, Frontend Masters, Pluralsight, GitHub awesome-cloudflare); free YouTube content is comprehensive; Cloudflare ships official deep-guide content displacing third-party offerings.

## Confirmer
Reddit /r/cloudflare and Cloudflare Discord questions show recurring patterns the docs don't answer; existing comparable Cloudflare-Workers content is shallow / outdated; comparable AWS Lambda books prove price tier.

## Payout
$49–99 one-time × 100–300 sales = $5–30k. Plausible to clear $10k with focused launch.

## Cost-to-falsify
Low-medium. ~4 hours: book/course competitor scan, /r/cloudflare audit, Cloudflare developer Discord scan, comparable Lambda product pricing.

## Bottom-line
**Strong candidate.** Audience is technical and willing-to-pay; distribution is straightforward (HN, /r/cloudflare, Cloudflare Discord). Risk: time to author quality content in 14 days. Advance to Phase 2 high priority.
