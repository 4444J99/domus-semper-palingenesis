# H-13 — GitHub Security Advisory curated vendor digest

**Category:** Newly-public information + aggregation gap

## Claim
GHSA feed is public and high-volume. Enterprise security teams want vendor-curated advisories ("only what affects vendors I use") delivered as a daily digest. A SaaS at $99–499/mo for SMB security teams, or a free-tier with paid alerts, converts.

## Why missed
- Snyk, Dependabot, Socket cover automatic patching but don't deliver editorial digests.
- The "human-readable, scoped" digest isn't a flashy product.
- Selling B2B SaaS to security teams in 30 days is hard without warm intros.

## Falsifier
Dies if: existing free email digests (e.g., "tl;dr sec," "Risky Biz") cover the niche; SMB security teams use Snyk/Dependabot directly and don't want a digest; sales cycle for security tools is structurally >30 days.

## Confirmer
Security newsletter market shows paid offerings; vendor-scoped curation isn't covered by incumbents; security pros explicitly ask for it.

## Payout
$99–499/mo × 5–25 subs in 30 days = $500–12k/mo. Possible to clear $10k if a single enterprise-tier customer signs.

## Cost-to-falsify
Medium. ~5 hours: existing security-newsletter scan, Snyk/Dependabot feature audit, B2B security sales-cycle reality check.

## Bottom-line
**Marginal for 30 days.** B2B SaaS sales cycle is structurally too long. Better as a 90-day play. Lower priority for Phase 2.
