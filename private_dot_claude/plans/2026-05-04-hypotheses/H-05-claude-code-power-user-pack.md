# H-05 — Claude Code power-user pack (skills/hooks/agents/commands)

**Category:** Documentation gap + ecosystem timing arbitrage

## Claim
Claude Code (the CLI) ships with hooks, skills, slash commands, agents, MCP server config, and a plugin system. Documentation exists but is scattered; most users use ≤20% of capabilities. A curated Power-User Pack — a downloadable / installable bundle of opinionated hooks, skills, agents, commands, plus a tutorial — converts at $29–79 to devs already paying $200/mo for Max plans.

## Why missed
- Users discover capabilities one at a time, by accident.
- Anthropic's docs are reference-style, not "set up your dream Claude Code in one hour."
- Most Claude Code customizations live in private dotfiles (no commercial product surface).
- Operator's own dotfile setup (chezmoi-managed `~/.claude/`) IS a candidate template — meta-arbitrage, but conflicts with "build outside operator's pre-built scaffold." Need a clean-room version.

## Falsifier
Dies if:
- A free GitHub repo (e.g., "awesome-claude-code") already provides a comprehensive setup with thousands of stars.
- Claude Code user base is too small / too price-insensitive for a $29–79 pack to gross $10k.
- Anthropic ships an official "starter pack" displacing third-party offerings.
- Distribution: with no audience, getting the pack in front of Claude Code Max users is hard.

## Confirmer
True if:
- "How do I configure Claude Code for X" Reddit/HN/Discord threads exist regularly.
- Existing community Claude Code resources are awesome-style lists, not opinionated bundles.
- Claude Code Max pricing ($200/mo) implies user base willing to spend on tooling.
- Comparable products (Vim/Tmux/Emacs setup-as-a-pack) have proven $29-99 sales channels.

## Payout
- $29–79 one-time × 200–500 sales = $6–40k. Clears $10k with 200 sales at $50.
- Subscription tier ($9/mo) for ongoing updates as Claude Code evolves.
- Bundled video walkthrough at $99–149 (higher conversion to power users).

## Cost-to-falsify (Phase 2 effort)
Low. ~3 hours: GitHub awesome-claude-code search, Reddit /r/claudecode + /r/anthropic + relevant Discord scan, comparable-product pricing check (Gumroad / Lemon Squeezy bundles).

## Bottom-line viability
**Strong candidate.** Audience is concentrated (Claude Code Max subscribers), willing-to-pay is proven ($200/mo plans), distribution is direct (post on relevant subreddits, HN). Single risk: Anthropic-ship displacement. Compounds with operator's own deep Claude Code expertise. **Top tier** — advance to Phase 2 with high priority. CAVEAT: must build clean-room (no operator's private chezmoi assets), respecting `feedback_pure_business_income_pov.md`.
