# H-18 — Claude Code enterprise onboarding kit + paid implementation

**Category:** Documentation gap + ecosystem timing

## Claim
Enterprises adopting Claude Code (post-Anthropic enterprise sales push) need: hooks for security/audit, MCP server allowlist policy, settings.json organizational defaults, internal CLAUDE.md templates, integration with existing CI/CD, training materials. A paid kit ($499–1999 per company) + optional implementation call ($500/hr) converts to companies already buying Claude Code Max for teams.

## Why missed
- Anthropic provides Claude Code but not opinionated enterprise rollout content.
- Most consultants targeting "AI rollout" target the model layer, not the IDE/CLI tool.
- Enterprise buyers want done-for-them, not docs.

## Falsifier
Dies if: Anthropic ships official enterprise rollout playbook displacing third parties; enterprises won't buy from a sole proprietor on a $499–1999 product; Claude Code adoption in enterprises is too early.

## Confirmer
LinkedIn / Anthropic forum / company blog posts about Claude Code rollouts; Anthropic enterprise sales velocity proven by recent case studies; comparable enterprise IDE / DevTool kits priced in target range.

## Payout
$499–1999 × 5–25 sales = $2.5–50k. Plausible to clear $10k with 5–20 sales.

## Cost-to-falsify
Medium. ~5 hours: enterprise Claude Code adoption signal scan, comparable IDE-rollout-kit pricing, LinkedIn outreach feasibility test.

## Bottom-line
**Strong candidate.** Audience is enterprise + willing-to-pay + Claude Code adoption is hot. Single sale $499+ accelerates revenue. Advance to Phase 2 high priority.
